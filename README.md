# Farmer Help App 🌾

Farmer Help App is a simple web-based application designed to support farmers with useful agricultural information in one place. It provides crop suggestions based on soil type, weather updates, fertilizer guidance, loan calculation, market crop prices, and pest control tips.

## Unique Features
- Soil-based Crop Recommendation System  
- Weather Information Display  
- Fertilizer and Pest Control Tips  
- Farmer Loan Interest Calculator  
- Live Crop Market Price Section  
- Interactive UI using jQuery Effects (Slide, Fade Toggle)  
- Simple, user-friendly and responsive design  

## Technologies Used
HTML, CSS, JavaScript, jQuery
